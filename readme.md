# 🛍️ VeImports — Site Oficial

E-commerce completo de camisas masculinas desenvolvido para um cliente real. Projeto com múltiplas páginas por categoria, design e identidade visual criados do zero, integração com WhatsApp e deploy ativo.

---

## 📋 Sobre o Projeto

Site oficial da VeImports, loja especializada em camisas masculinas. Desenvolvido com foco em experiência de navegação, organização por categorias e conversão de vendas via WhatsApp.

Todas as imagens dos produtos, o design e a identidade visual foram criados pelo desenvolvedor. O projeto evoluiu a partir de uma versão beta, com reestruturação completa do layout e adição de páginas por categoria.

---

## ✅ Funcionalidades

- Página inicial com vitrine de produtos em destaque
- Navegação por categorias: Camisas Europa, Nacionais e Seleções
- Integração com WhatsApp para conversão direta de vendas
- Layout responsivo para mobile e desktop
- Transições animadas e sliders de produtos
- Design e identidade visual próprios

---

## 🗂️ Estrutura de Páginas

| Página | Descrição |
|---|---|
| `index.html` | Página inicial — vitrine geral |
| `camisa europa.html` | Categoria camisas europeias |
| `camisa nacional.html` | Categoria camisas nacionais |
| `camisa seleção.html` | Categoria camisas de seleções |
